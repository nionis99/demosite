# DEMOSITE

Full responsive HTML - CSS template from psd on React.

## run
``npm start``

## build
``npm build``

## file structure
```
--src/
  --components/
        --App/
        --Layout/
        --Pages/
        --...
  --styles/
        --App/
        --Layout/
        --Pages/
        --...
  --images/
    index.js
    Routes.js
  --...
```
